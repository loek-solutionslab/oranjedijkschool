# Oranje-Dijkschool — demo rebuild

A modern, conversion-focused rebuild of [oranjedijkschool.com](https://www.oranjedijkschool.com/), built by [Solutions Lab](https://solutionslab.nl).

Stack: Vite + TypeScript + Tailwind CSS v4 (multi-page).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Railway-ready via `railway.json`. Push to GitHub and connect the repo in Railway.

---

The audit page at `/pages/solutionslab.html` shows the analysis behind this rebuild.
